# 📊 Super Store Sales Dashboard (Power BI)

## 🔍 Project Overview
This project analyzes retail Super Store sales data using Microsoft Power BI. It includes data preprocessing, analysis, and interactive dashboard visualization.

## 📁 Files Included
- CA2.pbix → Power BI Dashboard file
- PowerBI.pdf → Project Report
- Dashboard Screenshot

## 📊 Key Insights
- Technology is the top-performing category
- Consumer segment contributes highest revenue
- California and New York are top states
- Sales peak in Q4
- COD is the most used payment mode

## 🛠 Tools Used
- Microsoft Power BI
- Power Query Editor

## 📌 Features
- Interactive dashboard
- Cross-filtering visuals
- KPI cards
- YoY analysis
